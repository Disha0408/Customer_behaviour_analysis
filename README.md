# Customer_behaviour_analysis
Data Analtics Project showing Customer behaviour analysis using Python,sql and power BI

Overview
A complete data analytics project analyzing customer shopping behaviour using Python, SQL (PostgreSQL), and Power BI. It includes data loading, EDA, cleaning, SQL querying, and dashboard creation to uncover key trends in customer purchases.

Dataset
Customer purchase data containing: demographics, product details, purchase amount, review rating, discounts, payment method, and purchase frequency.

Tools Used
->Python (Pandas, NumPy, Matplotlib)
->PostgreSQL + pgAdmin
->Power BI
->Jupyter Notebook / VS Code

Project Steps
1.Load dataset in Python
2.Perform EDA and summary statistics
3.Clean, transform, and prepare data
4.Load cleaned data into PostgreSQL
5.Run SQL queries for deeper insights
6.Build an interactive Power BI dashboard

Results
->Identified top-selling categories and customer segments
->Observed spending patterns by age, season, and location
->Highlighted effects of discounts, subscription status, and ratings

How to Run

->Clone the repo
->Install Python dependencies
->Run Jupyter Notebook for EDA
->Connect PostgreSQL and execute SQL scripts
->Open Power BI .pbix file and refresh data
